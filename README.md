# BullshitGenerator-Flask

This is a Flask implementation of BullshitGenerator base on:

https://github.com/aetherwu/

which is base on:

https://github.com/menzi11/BullshitGenerator

originally.


## Run in Python 3.6

```
virtualenv flask
flask/bin/pip install -r requirements.txt
flask/bin/python debug.py
```

Deploy with tornado:

> flask/bin/python run.py

## Get data

Demo:

http://0.0.0.0:5002/g

http://0.0.0.0:5002/g?xx=今晚吃啥
